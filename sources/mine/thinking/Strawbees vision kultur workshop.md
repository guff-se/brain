What happened today?

Sjukt fina människor. Allt verkar så enkelt. Samtalet om värderingar var fint, men landade inte i något. Kommer nog bli bra med återkommande övningar. Prototypverkstaden var över förväntningar (igen)… varför litar jag inte på att det kommer att vara så? Två förslag: Bygga strawbees org-chart, samt veckofrukostar. Sjukt bra början.

Startade inte med checkin… det borde jag gjort.

Feedback-utbildning med Nina. Bra grej som borde bokas in.

How did it make me feel?

Fan att jag inte sovit. Så jädra trött idag. Hade lätt kunnat få en timma till om jag inte tagit upp datorn.

Det som var fint var att precis som gårdagens möte (kulturbryggan), så kunde jag bara dra mina 5 bulletpoints “det här skall vi göra idag” och det sålde direkt. Skönt att kunna känna att jag skapar det förtroendet för processen. När jag säger “detta gör vi”, så är folk med.

Teal landade fint. Gjorde min första beskrivning utifrån utvecklingsnivåer. Blev faktiskt riktigt bra. Gillar inte alls att använda färger, framför allt på svenska med orden “amber” och “teal” som inte översätter så bra.

Jag gillade att så många gillade teal-tankarna. Känns helt rätt att börja med detta nu.

What did I learn?

- Starta med Checkin
- Lita på att folk litar på mig att guida processen
- Lita på att jag faktiskt klarar att guida processen.
- 3 whys funkade fint.
