# skola beksrivning föreläsning  
  
Vad är skolans roll när läraren ersatts av AI?   
Gustaf berättar om hur teknik inte bara förändrar hur vi tar till oss information, utan hur vi ser på varan som människor. Det är teknologin som genom tiderna definierat relationer, tillhörighet och makt. Hur känns det för människan att bli digitaliserad? Hur kan vi använda de nya verktygen för att skapa närhet och tillhörighet? Hur kan de användas för att skapa konflikt och utanförskap?  
Och vad blir egentligen skolans roll lärandet är livslångt? När elever i allt högre grad sitter i klassrum och vid middagsbord med tillgång till perspektiv som deras lärare och föräldrar inte har. Kanske är detta början på ett nytt välfärdsprojekt, en lika omvälvande och ambitiös förändring som när den skapade allmänna folkskolan skapades.
