Hej!

Det var fantastiskt trevligt att träffas igår. På vägen hem så sade jag och Malin till varandra att det kändes nästan för bra, som om det måste funnits någon hake som vi missat. Vi har ju såklart mycket jobb och många hinder framför oss, men just nu känns det i alla fäll väldigt bra!

Enl. ök. så kommer här våra ganska lösa ramar på hur vi skulle kunna göra detta...

"Version 1" känns efter vårt samtal som det minst intressanta. Idén är helt enkelt att vi samlar ihop ett gäng och köper lägenheter tillsammans, precis som era vanliga “kompisköp". Det kanske är ett alternativ om annat fallerar, men jag tycker vi skall börja sikta lite högre.

Version 2

-
Lägenheter ser ut som de gör.

-
Gränslandet tar på sig rollen som “mäklare", med ett säljmål på x antal lägenheter, t.ex. en hel uppgång eller en hel fastighet.

-
Gränslandet har ett exklusivt säljfönster som öppnar före normalt planerad säljstart. Eftersom vi säljer via nätverk så kan detta eventuellt pågå parallellt med en tidigare etapp (förutsatt att vi hittar en modell för att hantera kölistan).

-
Det som inte är sålt inom detta fönster övergår till Atriums vanliga säljkanaler, ev. parallellt med Gränslandets fortsatta arbete.

-
Gränslandet erhåller motsvarande mäklararvode för sålda lägenheter.

Version 3

-
Som version 2, fast vi ger också input till byggprocessen, för att t.ex. lägga till gemensamhetsutrymme eller påverka storlekar på lägenheter.

-
Detta skulle antingen kunna göras som ett lite större projekt där vi verkligen är med och påverkar, inför den 3e byggetappen.

-
Alternativt får vi förhålla oss till de mindre ändringar som ändå kan vara möjliga efter att det ansökts om bygglov för etapp 2.

Det vi helst skulle vilja göra nu är att utforska detta sista alternativ, alltså ett sätt att passa in i etapp 2, utefter de förutsättningar som finns. Det känns som att vi har en del tid på oss om vi skulle välja att gå på etapp 3, så därför kan vi ju börja med att lägga fokuset på etapp 2 och se hur det går.

Vi kommer ses igen den 25 oktober. Tills dess kommer vi att börja sondera terrängen hos de som var medlemmar i vårt tidigare projekt för att se vilket intresse som finns. Eftersom det tidigare projektet gällde villor som kostade 5 miljoner och uppåt, så kan vi redan nu säga att det intresse som ev. finns främst kommer att vara för 3or eller större lägenheter. Dock har vi folk i vårt utökade nätverk som vi vet vill ha mindre och som inte varit med i vårt projekt på grund av storeken, så det kan givetvis vara möjligt för oss att hitta köpare även för mindre lägenheter.

Vi hoppas att ni kanske till nästa möte kan ta fram ett par egna “versioner”, alternativ för hur vi skulle kunna gå vidare. Några frågor som spontant finns i våra huvuden är:

-
Hur ser planerna för etapp 2 ut? Vilka lägenheter kommer finnas? Kan vi redan nu börja titta på specifika byggnader eller uppgångar?

-
Vad sätter kölistan för begränsningar?

-
Hur ser tidsplanen ut? När skulle vi kunna börja sälja, när måste vi vara klara?

-
Vilka andra hinder kan finnas?

-
Vad är nästa steg?

Värt att tillägga är att vi tidigare hade tänkt göra allt själva, så vi har i vårt gäng allt från fastighetsutvecklare och arkitekter till ekonomer och jurister. Behövs ytterligare kompetens, eller arbetsinsats från vårt håll så går det alltså att lösa.
